🚀 NASA Mission Explorer
🌌 About

NASA Mission Explorer is a simple web app that lets users explore NASA’s Astronomy Picture of the Day (APOD). Users can select any date to see what the universe looked like on that day — even their birthday.

🎯 Features
📅 Date picker to view APOD for any day
🎂 Birthday explorer
🖼️ Full-screen image view
🔘 HD image toggle
🌙 Dark space-themed UI
🔌 API

Using NASA APOD API:
https://api.nasa.gov/

🛠️ Tech Stack
HTML
CSS
JavaScript (Vanilla JS)
Fetch API
⚙️ Setup
Clone the repository:
git clone https://github.com/your-username/nasa-mission-explorer.git
cd nasa-mission-explorer
Open the project:
Simply open index.html in your browser
(No build tools required)
Add your API key:
Get it from https://api.nasa.gov/
Replace DEMO_KEY in your JavaScript file with your API key
